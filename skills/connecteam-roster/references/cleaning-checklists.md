## 4. AIRBNB TURNOVER CHECKLIST

### Entryway
- [ ] Disinfect door handles (inside and outside)
- [ ] Sweep/vacuum/mop floors
- [ ] Wipe light switches
- [ ] Check for cobwebs
- [ ] Spot clean walls/scuff marks
- [ ] Ensure doormat is clean

### Kitchen
- [ ] Wipe all countertops and splashback
- [ ] Clean inside microwave, oven, fridge
- [ ] Clean stovetop
- [ ] Wipe all cupboard fronts
- [ ] Clean sink, taps, drain
- [ ] Run empty dishwasher or clean by hand
- [ ] Take out bin and disinfect bin lid
- [ ] Restock essentials (sponge, dish liquid, paper towels)
- [ ] Sweep and mop floors

### Bedrooms
- [ ] Strip beds completely
- [ ] Replace with fresh laundered linen and pillowcases
- [ ] Make beds to hotel standard
- [ ] Wipe bedside tables and lamps
- [ ] Dust ceiling fans and vents
- [ ] Check under beds for left-behind items
- [ ] Vacuum thoroughly including corners

### Bathroom(s)
- [ ] Disinfect toilet, sink, shower, taps
- [ ] Polish mirrors and glass
- [ ] Replace all towels and bath mats
- [ ] Restock toiletries (toilet paper, soap, shampoo)
- [ ] Empty bathroom bin and sanitise
- [ ] Mop floor with disinfectant

### Living Room
- [ ] Vacuum carpets / mop hard floors
- [ ] Dust all surfaces (tables, TV stand, shelves)
- [ ] Clean remotes and electronic devices
- [ ] Fluff and straighten cushions
- [ ] Disinfect light switches and door handles
- [ ] Check for rubbish or leftover items

### Outdoor Areas
- [ ] Sweep balcony/patio
- [ ] Wipe outdoor furniture
- [ ] Remove any rubbish
- [ ] Check outdoor lighting works

### Final Checks
- [ ] Walk through each room with fresh eyes
- [ ] Check all lights and appliances work
- [ ] Ensure WiFi is working
- [ ] Set thermostat/AC to comfortable temp
- [ ] Leave welcome note/info pack visible
- [ ] Lock all windows and set entry code
- [ ] Take photos for records
- [ ] Report any damages or maintenance needed
