# Social Weekly Planner

## Purpose
Create a practical 7-day plan (FB + IG) with:
- pillar rotation
- local hook adaptation
- CTA rotation

## Input
- Start date (YYYY-MM-DD)
- Priority suburb
- Typical property size (bedrooms)

## Command
```bash
python3 scripts/social_growth_weekly_planner.py --start-date 2026-02-23 --suburb Liverpool --bedrooms 3 --out references/social-growth/weekly-plan.md
```

## Output
- Markdown calendar with day-by-day content instructions
