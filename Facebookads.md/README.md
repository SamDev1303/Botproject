# Facebook Ads + Social Growth Pack (Clean Up Bros)

This folder stores your live social/ad system for Clean Up Bros.

## Included
- `social-growth-strategy.md` — full strategy
- `daily-ops-checklist.md` — daily execution checklist
- `skill-test-proof.md` — test results summary
- `weekly-plan.md` — 7-day content schedule
- `next-7-posts-ready.md` — ready-to-publish captions
- `ad-ab-test-plan.md` — practical ad test matrix
- `references/` — hooks + CTA matrix + sample AB dataset

## Usage
1. Publish from `next-7-posts-ready.md`
2. Run ad tests from `ad-ab-test-plan.md`
3. Review weekly and iterate.
